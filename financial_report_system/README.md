## 📋 项目简介

本系统是一款面向银行和金融机构的企业信贷风险分析工具，通过上传企业财务报表，自动进行多维度风险评估，生成专业的信用风险评估报告。

### 核心功能

- 📊 **三表全量解析**：资产负债表、利润表、现金流量表完整提取
- 🔍 **双引擎读取**：支持 .xls（xlrd）和 .xlsx（openpyxl）格式
- ✅ **财务勾稽校验**：期末现金一致性、会计恒等式、利润含金量验证
- 📅 **期间类型识别**：年报/季报/月报自动识别 + 年化因子折算
- 🏦 **行业差异化**：支持6种行业分类，各维度权重可配置
- 📈 **六维评分体系**：偿债能力、盈利能力、现金流、营运能力、税务合规、造假预警
- 🎯 **Beneish M-Score**：财务造假预警模型（8因子完整计算）
- 📋 **知识库规则**：内置16条风控规则，自动触发预警
- 🏭 **行业基准参照**：内置制造业、担保/金融服务等行业的参考指标
- 🌐 **异步任务处理**：支持长时间分析任务的进度轮询
- 📊 **实时进度显示**：前端展示分析进度，无需等待
- 🎨 **现代化界面**：渐变卡片设计，流畅动画体验
- 📥 **PDF导出**：生成完整PDF报告，可直接下载

## 🚀 快速开始

### 方式一：Docker 部署（推荐）

```bash
cd docker
docker-compose up -d
# 访问 http://localhost:8000
```

### 方式二：本地运行

**Linux/Mac:**
```bash
chmod +x start.sh
./start.sh
```

**Windows:**
```bash
start.bat
```

### 方式三：直接运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
uvicorn app.main:app --reload
# 访问 http://localhost:8000
```

### 方式四：命令行分析（CLI）

```bash
cd risk_engine

# 分析单期数据
python main.py --files ../data/2025.12.xlsx --company "企业名称" --industry 制造业

# 分析多期数据（自动跨期对比）
python main.py --files ../data/2025.12.xlsx ../data/2026.3.xlsx --company "企业名称" --industry 制造业

# 扫描目录
python main.py --dir ../data --company "企业名称" --industry 担保/金融服务
```

## 🔌 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 首页 |
| `/health` | GET | 健康检查 |
| `/api/analyze` | POST | 同步分析 |
| `/api/analyze-async` | POST | 异步分析（返回task_id） |
| `/api/task/{task_id}` | GET | 查询任务状态 |
| `/api/report/{filename}` | GET | 下载报告 |
| `/api/system/status` | GET | 系统状态 |
| `/api/cleanup` | POST | 清理过期报告 |

### API 使用示例

```bash
# 同步分析
curl -X POST http://localhost:8000/api/analyze \
  -F "company_name=测试公司" \
  -F "industry=担保/金融服务" \
  -F "files=@财务报表.xlsx"

# 异步分析
curl -X POST http://localhost:8000/api/analyze-async \
  -F "company_name=测试公司" \
  -F "industry=担保/金融服务" \
  -F "files=@财务报表.xlsx"

# 查询任务状态
curl http://localhost:8000/api/task/{task_id}
```

## 🔬 核心算法

### 三表勾稽校验

- **期末现金一致性**：资产负债表货币资金 = 现金流量表现金期末余额
- **会计恒等式**：资产 = 负债 + 所有者权益
- **利润含金量**：经营现金流/净利润 > 50%

### Beneish M-Score（8因子模型）

| 因子 | 含义 |
|------|------|
| DSRI | 应收账款指数 |
| GMI | 毛利率指数 |
| AQI | 资产质量指数 |
| SGI | 营收增长指数 |
| DEPI | 折旧指数 |
| SGAI | 销管费用指数 |
| TATA | 应计利润比率 |
| LVGI | 财务杠杆指数 |

**判别规则：**
- M-Score < -2.22 → 财务造假可能性低
- M-Score > -1.78 → 财务造假可能性高

### 期间类型识别

| 类型 | 识别方式 | 年化因子 |
|------|----------|----------|
| 年报 | 12月，比值<0.20 | 1.0 |
| 季报 Q1 | 3月，比值≈0.25 | 4.0 |
| 季报 Q2 | 6月，比值≈0.50 | 2.0 |
| 季报 Q3 | 9月，比值≈0.75 | 4/3 |
| 月报 | 其他 | 12/month |
